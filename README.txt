How to use:
1.Create two folders in the same folder as main.py, one named "temp" and second named "temp2"
2.Download this texture pack https://www.curseforge.com/minecraft/texture-packs/vanilladefault/download/3669921 (or use any other you want it must be 1.18.2)
3.Unzip the downloaded pack, put it in the same folder as main.py
4.Put the texture pack name in .env file (use notepad)
5.Run main.py using IDLE
